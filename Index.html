Inclusive Gamification Framework

Ein Prototyp zur Verbindung von Gamification, Barrierefreiheit und User Experience

1. Analyse-Schicht

In dieser Schicht wird geprüft, welche Barrieren in einer Anwendung entstehen können. Dazu gehören zum Beispiel Zeitdruck, rein visuelles Feedback oder unklare Bedienwege.

Checkliste




2. Adaptions-Schicht

Hier werden problematische Gamification-Elemente angepasst. Ziel ist nicht, Gamification zu entfernen, sondern sie inklusiver zu gestalten.

Timer → elastische Zeitvorgaben

Leaderboard → persönlicher Fortschritt

Badge → visuelles, textliches und akustisches Feedback

3. Validierungs-Schicht

Am Ende wird geprüft, ob die Anwendung wirklich zugänglicher und angenehmer nutzbar ist. Dazu können Walkthroughs, Nutzerfeedback oder Fragebögen eingesetzt werden.

Was wurde durch das IGF verbessert?

Der feste Timer wurde entfernt. Statt eines öffentlichen Rankings wird persönlicher Fortschritt angezeigt. Das Badge wird nicht nur visuell vermittelt, sondern zusätzlich erklärt. Dadurch bleibt Gamification erhalten, wirkt aber weniger ausschließend.
